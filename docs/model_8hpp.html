Detailed Description

RiverSim - is big program with a lot of parameters. Program handles geometry, boudary conditions, mesh generation, FEM solvers etc. And each this module has a lot of parameters.

These parameters can be specified by JSON file(io.hpp) or through program options(River::process_program_options).
