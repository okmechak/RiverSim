Detailed Description

File holds two classes: River::BranchNew and River::Rivers. As you can see from picture - River::BranchNew is simple array joined points - piecewice line, aproximation of real river branch. River::Rivers on other hand consist from combination of Branches. Bifurcation point - where two rivers unions into one river. So each branch has its subbranches. Structure of branches and sub branches is represented here in class River::Rivers.
